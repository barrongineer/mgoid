# mgoid

A simple cli to generate and validate Mongo ObjectID's.

Usage:
```
github.com/barrongineer/mgoid                                                                                                                                                                                                                         
▶ mgoid new
5a4e5d29768a4174ea47d06f

github.com/barrongineer/mgoid                                                                                                                                                                                                                         
▶ mgoid validate 5a4e5d29768a4174ea47d06f
valid

github.com/barrongineer/mgoid                                                                                                                                                                                                                         
▶ mgoid validate something
invalid

```
